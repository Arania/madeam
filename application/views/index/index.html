Setup

1. Setup your database and configure MOD_REWRITE based on your server setup:

/config/environment.php

2. To edit the layout of this page open the following file:

/application/views/_layouts/standard.html

3. To edit this page's content (the view) open the following file:

/application/views/index/index.html

4. To edit the controller that executes PHP before this page is rendered edit the following file:

/application/controllers/index_controller.php
